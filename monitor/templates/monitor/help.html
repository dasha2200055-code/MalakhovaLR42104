Об угрозе УБИ.196

УБИ.196 - «Угроза контроля вредоносной программой списка приложений, запущенных на мобильном устройстве».

Злоумышленник, используя вредоносную программу, может получать информацию о том, какие приложения активны в данный момент. Эти данные могут использоваться для кражи персональных данных, слежки за действиями пользователя или для целенаправленного воздействия на конкретные приложения.

Данная угроза реализуется из-за недостаточности мер антивирусной защиты и отсутствия контроля за разрешённым ПО.

Как работает система защиты

Система анализирует каждое приложение по трём критериям и рассчитывает уровень риска.

📊

Критерий 1

Частота запросов

Если приложение запрашивает список запущенных программ более 50 раз — это подозрительно

🔑

Критерий 2

Доступ к API

Если приложение имеет доступ к системным функциям — оно может следить за другими программами

❓

Критерий 3

Неизвестное происхождение

Если приложение не из доверенного списка (содержит «Unknown» в названии)

Правила принятия решений

Уровень рискаСтатусДействие
0–1 балл РАЗРЕШЁН Приложение считается безопасным
2–3 балла ЗАБЛОКИРОВАН Приложение признаётся вредоносным, рекомендуется удалить

Рекомендации пользователю

  1. Проверьте список заблокированных приложений на странице «Главная».
  2. Изучите графики на странице «Графики» для наглядного анализа.
  3. Приложения с пометкой «Неизвестное» проверьте вручную.
  4. Приложения с высоким риском и доступом к API рекомендуется удалить.
  5. После удаления перезапустите анализ кнопкой «Обновить анализ».
